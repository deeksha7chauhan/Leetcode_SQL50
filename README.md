**IFNULL(..., NULL)**:Wraps the subquery to return NULL if the subquery returns no rows explicitly
The IFNULL() function returns a specified value if the expression is NULL.
SYntax:-
SELECT IFNULL((
    -- Subquery here
), NULL) AS num;
